Welcome to Rainmire

If you see this page, NGINX is successfully serving your HTML file.

If you were expecting to see a different application, here are a few things to check:

  • Ensure that the server_name in your NGINX server block matches the domain you are accessing.
  • Check that the location block for the path you're trying to access is properly configured to proxy requests to the correct application.
  • Make sure that your DNS settings are correctly pointing to this server's IP address.
  • Ensure that your port forwarding settings are correctly configured if you're accessing this server from outside your local network.

This is a diagnostic page to help you troubleshoot your NGINX configuration.
